## ğŸ¨ Customization

### Colors
Edit the CSS variables in `:root` to customize the color scheme:

```css
:root {
  --bg: #030712;           /* Background */
  --brand: #06b6d4;        /* Primary accent (Cyan) */
  --brand-2: #8b5cf6;      /* Secondary accent (Purple) */
  --brand-3: #ec4899;      /* Tertiary accent (Pink) */
  --text: #f1f5f9;         /* Text color */
  --muted: #94a3b8;        /* Muted text */
}
```
